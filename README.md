# Healfast India Website (Frontend)

Modern, responsive healthcare website for **Healfast India**.

## Project Structure

- `frontend/` → Next.js + Tailwind CSS + Framer Motion

## Frontend Features

- Home, Services, About, Blog, Contact pages
- Sticky navbar + smooth scrolling
- Responsive modern UI (medical trust theme)
- WhatsApp floating button + mobile call button
- Reusable components (cards, buttons, sections)
- SEO-friendly metadata

## Run Locally

### Frontend

1. Open terminal in `frontend/`
2. Install deps: `npm install`
3. Start: `npm run dev`

Frontend runs on `http://localhost:3000`

---

You can now edit text, images, contact details, and add real certificates/blog content.
